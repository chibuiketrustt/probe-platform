# PROBE Platform

Production financial infrastructure foundation built with:
- Next.js (App Router)
- TypeScript
- Tailwind
- Supabase
- Vercel

## Current stage

This repository currently contains **architecture foundation only**:
- Scalable folder structure
- Environment variable system
- Supabase client separation (browser/server/admin)
- Global typing setup
- Centralized config layer

See detailed design in [`docs/architecture.md`](docs/architecture.md).
